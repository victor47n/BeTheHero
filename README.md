## 🔖 Como Executar

#### Clonando o projeto
```sh
git clone https://github.com/victor47n/BeTheHero.git
cd BeTheHero
npm install
```
#### Iniciando Backend
```sh
cd backend
npm start
```
#### Iniciando Frontend
```sh
cd frontend
npm start
```
#### Iniciando mobile
```sh
cd mobile
npm start
```
